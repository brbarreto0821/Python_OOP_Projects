Setup:

Open the command line and then enter:

$ cd ..\MachineLearningRGB
$ python colors.py

Select the color that shows up as many times as you want. Once done,
enter this in the command line:

$ python test.py

Select yes or no if the color that shows up is the color that appears. Once done, you can enter this in the command line to see the color data that you created from colors.py:

$ python output.py
